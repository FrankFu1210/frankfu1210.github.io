                                  Nche TubeLeaks 5 years ago edited

                                  I used this plugin on my webpage: www.blood-and-stool.com, and it worked perfectly for me. Here are some tips.
                                  (1.) Load the required jQuery library and bootstrap stylesheet:
                                  <script src="js/bootstrap.min.js"></script>
                                  <script src="js/jquery.mycart.js"></script>
                                  <script src="js/jquery-2.2.3.min.js"></script>
                                  <script src="js/jquery.mycart.min.js"></script>
                                  <link href="css/bootstrap.min.css" rel="stylesheet">

                                  NB: Only use the version of bootstrap (bootstrap.min.css v3.3.6 and bootstrap.min.js v3.3.6) that are found in the contents of the download file.

                                  (2.) use this ''div'' to insert the badge and badge-notify to your navbar:
                                  <li></li>

                                  (3.) Use this CSS:

                                  .badge-notify {
                                  background: #6394f8;
                                  }
                                  .badge {
                                  line-height: 10px;
                                  position: relative;
                                  }
                                  .modal {
                                  margin-top: 100px;
                                  z-index: 9999;
                                  }
                                  .table {
                                  display: block;
                                  padding: inherit;
                                  }

                                  I hope this would help you. Nche
